# Conceptual Framework

## กรอบแนวคิดของแบบจำลองความเสี่ยงน้ำท่วมในจังหวัดสุโขทัย

---

# 1. คำถามวิจัย (Research Question)

พื้นที่ใดในจังหวัดสุโขทัยมีความเสี่ยงต่อการเกิดน้ำท่วมสูง โดยพิจารณาจากปัจจัยด้านสิ่งแวดล้อม ได้แก่ ปริมาณฝน ระดับความสูงของพื้นที่ ความลาดชัน ระยะห่างจากแม่น้ำ และสภาพพืชพรรณ

คำถามนี้สามารถตอบได้โดยการสร้าง **แบบจำลองเชิงพื้นที่ (Spatial Model)** เพื่อแสดงการกระจายตัวของความเสี่ยงน้ำท่วมในรูปแบบของแผนที่

---

# 2. แนวคิดของแบบจำลอง

การเกิดน้ำท่วมไม่ได้เกิดจากปัจจัยเพียงตัวเดียว แต่เกิดจากการรวมกันของหลายปัจจัยทางกายภาพ เช่น

- ปริมาณฝน
- ลักษณะภูมิประเทศ
- ความลาดชัน
- ความใกล้ชิดกับแม่น้ำ
- สภาพการปกคลุมของพืชพรรณ

ดังนั้น การศึกษานี้จึงใช้วิธี **Multi-Criteria Spatial Modeling** ซึ่งเป็นการรวมปัจจัยหลายตัวเข้าด้วยกันเพื่อประเมินระดับความเสี่ยงน้ำท่วม

แบบจำลองใช้วิธี **Weighted Linear Combination (WLC)** ในการรวมค่าของแต่ละปัจจัย

---

# 3. ปัจจัยที่ใช้ในแบบจำลอง

| ปัจจัย | ความสัมพันธ์กับน้ำท่วม | เหตุผล |
|------|-----------------|--------|
| Rainfall | ฝนมาก → ความเสี่ยงสูง | ปริมาณฝนที่สูงทำให้เกิดการสะสมของน้ำและเพิ่มโอกาสการเกิดน้ำท่วม |
| Elevation | พื้นที่ต่ำ → ความเสี่ยงสูง | พื้นที่ที่มีระดับความสูงต่ำมักเป็นพื้นที่รับน้ำ |
| Slope | ความลาดชันต่ำ → ความเสี่ยงสูง | น้ำไหลออกจากพื้นที่ช้า |
| Distance to River | ใกล้แม่น้ำ → ความเสี่ยงสูง | มีโอกาสเกิดน้ำล้นตลิ่ง |
| NDVI | พืชพรรณมาก → ความเสี่ยงต่ำ | พืชช่วยดูดซับน้ำและลดการไหลบ่าของน้ำ |

---

# 4. การกำหนดน้ำหนักของปัจจัย

น้ำหนักของแต่ละปัจจัยถูกกำหนดตามความสำคัญทางภูมิศาสตร์และความสัมพันธ์กับการเกิดน้ำท่วม

| ปัจจัย | น้ำหนัก |
|------|-------|
| Rainfall | 0.25 |
| Elevation | 0.20 |
| Slope | 0.20 |
| Distance to River | 0.20 |
| NDVI | 0.15 |

น้ำหนักทั้งหมดรวมกันเท่ากับ **1.00** เพื่อให้สามารถนำไปใช้ในสมการ Weighted Linear Combination

---

# 5. สมการของแบบจำลอง

Flood Risk Index คำนวณจากสมการ

Flood Risk =  
0.25 × Rainfall +  
0.20 × Elevation +  
0.20 × Slope +  
0.20 × Distance to River +  
0.15 × NDVI

ค่าที่ได้จะอยู่ในช่วง **0 – 1** ซึ่งแสดงระดับความเสี่ยงของพื้นที่

---

# 6. การ Normalize ข้อมูล

เนื่องจากข้อมูลแต่ละประเภทมีหน่วยและช่วงค่าที่แตกต่างกัน จึงต้องทำ **Normalization** ก่อนนำมารวมในแบบจำลอง

วิธีที่ใช้คือ **Min-Max Normalization**

Normalized Value = (X − Min) / (Max − Min)

วิธีนี้ทำให้ค่าของทุกปัจจัยอยู่ในช่วง **0 ถึง 1** เพื่อให้สามารถนำมารวมกันในแบบจำลองได้อย่างเหมาะสม

---

# 7. การตรวจสอบแบบจำลอง (Validation)

การตรวจสอบแบบจำลองทำโดยการเปรียบเทียบพื้นที่ที่แบบจำลองระบุว่าเป็นพื้นที่เสี่ยงสูงกับข้อมูล **JRC Global Surface Water**

หากแบบจำลองมีความสมเหตุสมผล พื้นที่ที่มีความเสี่ยงสูงควรปรากฏใกล้กับพื้นที่ที่มีแม่น้ำหรือแหล่งน้ำถาวร

---

# 8. การวิเคราะห์ความไวของแบบจำลอง (Sensitivity Analysis)

เพื่อประเมินความไวของแบบจำลอง ได้ทำการปรับน้ำหนักของปัจจัย **Rainfall** เพิ่มขึ้นและลดลง **±20%**

จากนั้นนำมาคำนวณแบบจำลองใหม่และเปรียบเทียบผลลัพธ์กับโมเดลเดิม

การวิเคราะห์นี้ช่วยให้เข้าใจว่าแบบจำลองมีความไวต่อการเปลี่ยนแปลงของปัจจัยมากเพียงใด

---

# 9. สรุปแนวคิดของโมเดล

แบบจำลองความเสี่ยงน้ำท่วมที่ใช้ในการศึกษานี้พัฒนาขึ้นจากแนวคิดว่าการเกิดน้ำท่วมเป็นผลมาจากปัจจัยทางสิ่งแวดล้อมหลายประการที่ทำงานร่วมกัน ไม่ได้เกิดจากปัจจัยใดปัจจัยหนึ่งเพียงอย่างเดียว ดังนั้นการวิเคราะห์จึงใช้แนวทาง **Multi-Criteria Spatial Modeling** เพื่อรวมข้อมูลจากหลายปัจจัยที่มีความสัมพันธ์กับการเกิดน้ำท่วม

ในขั้นตอนแรก ข้อมูลปัจจัยด้านสิ่งแวดล้อม ได้แก่ ปริมาณฝน ระดับความสูงของพื้นที่ ความลาดชัน ระยะห่างจากแม่น้ำ และดัชนีพืชพรรณ ถูกนำมาประมวลผลและปรับค่าให้อยู่ในช่วงเดียวกันด้วยวิธี **Normalization** เพื่อให้สามารถเปรียบเทียบและรวมค่าระหว่างตัวแปรที่มีหน่วยแตกต่างกันได้อย่างเหมาะสม

จากนั้นค่าของแต่ละปัจจัยจะถูกนำมารวมกันด้วยวิธี **Weighted Linear Combination (WLC)** โดยกำหนดน้ำหนักของแต่ละปัจจัยตามความสำคัญทางภูมิศาสตร์ที่มีต่อการเกิดน้ำท่วม ผลลัพธ์ที่ได้คือ **Flood Risk Index** ซึ่งเป็นค่าดัชนีที่แสดงระดับความเสี่ยงของพื้นที่ในช่วงค่า **0 ถึง 1**

ค่าดัชนีดังกล่าวถูกนำไปสร้างเป็น **Flood Risk Map** เพื่อแสดงการกระจายตัวของพื้นที่ที่มีความเสี่ยงน้ำท่วมในจังหวัดสุโขทัยอย่างเป็นระบบ แผนที่นี้ช่วยให้สามารถระบุพื้นที่ที่มีความเสี่ยงสูงได้อย่างชัดเจน และสามารถใช้เป็นข้อมูลพื้นฐานในการวางแผนจัดการภัยพิบัติหรือการบริหารจัดการพื้นที่ในอนาคต
