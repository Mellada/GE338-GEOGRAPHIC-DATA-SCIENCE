# Mars-like Environments on Earth  
## การระบุพื้นที่บนโลกที่มีลักษณะแห้งแล้ง พืชพรรณต่ำ และดินโล่ง คล้ายพื้นผิวดาวอังคาร

---

# แนวคิดของโครงการ

หลายพื้นที่ของโลกกำลังเผชิญกับการเสื่อมโทรมของระบบนิเวศและการขยายตัวของพื้นที่แห้งแล้ง ทำให้เกิดภูมิประเทศที่มีพืชพรรณต่ำ ความชื้นต่ำ และดินโล่งจำนวนมาก พื้นที่ลักษณะนี้มีลักษณะคล้ายพื้นผิวของดาวอังคาร ซึ่งเป็นดาวเคราะห์ที่มีพืชพรรณน้อยและพื้นผิวแห้งแล้งเป็นส่วนใหญ่

โครงการนี้มีเป้าหมายเพื่อระบุพื้นที่บนโลกที่มีลักษณะดังกล่าว โดยใช้ข้อมูล Remote Sensing วิเคราะห์พื้นที่ที่มี

- พืชพรรณต่ำ
- ความแห้งแล้งสูง
- ดินโล่งจำนวนมาก

ผลลัพธ์จะเป็นแผนที่ที่แสดงการกระจายตัวของพื้นที่ที่มีลักษณะ Mars-like environments

---

# ส่วนที่ 1: Problem Statement

การเปลี่ยนแปลงสภาพภูมิอากาศและการใช้ที่ดินของมนุษย์ทำให้หลายพื้นที่ของโลกมีความแห้งแล้งเพิ่มขึ้น ส่งผลให้พืชพรรณลดลงและเกิดพื้นที่ดินโล่งจำนวนมาก พื้นที่ลักษณะนี้มีความคล้ายคลึงกับพื้นผิวของดาวอังคาร

การระบุพื้นที่ลักษณะนี้สามารถช่วยติดตามการขยายตัวของพื้นที่แห้งแล้ง การเสื่อมโทรมของระบบนิเวศ และการศึกษาสภาพแวดล้อมสุดขั้วบนโลก

การศึกษานี้จะใช้ข้อมูลดาวเทียมเพื่อวิเคราะห์พื้นที่ที่มีพืชพรรณต่ำและความแห้งแล้งสูงในระดับโลก

---

# ส่วนที่ 2: Research Questions

| Research Question | Dataset | Method |
|---|---|---|
| พื้นที่ใดบนโลกที่มีพืชพรรณต่ำและดินโล่งจำนวนมาก | MODIS NDVI | วิเคราะห์ค่าดัชนีพืชพรรณ |
| พื้นที่ใดมีความแห้งแล้งสูง | GPM precipitation + MODIS ET | วิเคราะห์ความแห้งแล้ง |
| พื้นที่ใดมีลักษณะ Mars-like environment | NDVI + Dryness index | รวมตัวแปรเพื่อสร้าง Mars-like index |

# ส่วนที่ 3: Data Inventory

| Dataset | Source | ใช้ตอบคำถาม | ใช้ทำอะไร | ข้อจำกัด |
|---|---|---|---|---|
| MODIS NDVI | MOD13Q1 | RQ1 | วิเคราะห์พืชพรรณ | Resolution 250 m |
| MODIS Evapotranspiration | MOD16A2 | RQ2 | วิเคราะห์การระเหย | เป็นค่าประมาณ |
| GPM Precipitation | NASA GPM | RQ2 | วิเคราะห์ปริมาณฝน | Spatial resolution ต่ำ |
| DEM | SRTM | Support | วิเคราะห์ภูมิประเทศ | ข้อมูล static |

---

# ตารางดัชนีที่ใช้ (Indices)

| ดัชนี | สูตร | ใช้วัดอะไร | เหตุผลที่ใช้ |
|---|---|---|---|
| NDVI | (NIR − RED) / (NIR + RED) | ความหนาแน่นพืชพรรณ | ใช้แพร่หลายในการศึกษาพืช |
| Dryness Indicator | precipitation / evapotranspiration | ความแห้งแล้ง | ใช้ระบุพื้นที่ dry region |
| Low Vegetation | 1 − NDVI | พื้นที่พืชน้อย | ใช้แทน bare soil |
| Mars-like Index | (Low vegetation + Dryness) / 2 | สภาพพื้นที่คล้ายดาวอังคาร | รวม vegetation และ dryness |

---

# ส่วนที่ 4: Methodology Flow

ขั้นตอนการวิเคราะห์

1. ดึงข้อมูลจาก Google Earth Engine
2. คัดเลือกช่วงเวลาและพื้นที่ศึกษา
3. คำนวณ NDVI เพื่อวิเคราะห์พืชพรรณ
4. วิเคราะห์ precipitation และ evapotranspiration เพื่อวัดความแห้งแล้ง
5. คำนวณ Mars-like Environment Index
6. สร้างแผนที่ Mars-like environments

---

# Workflow Diagram (Concept)

```
Satellite Data
     ↓
MODIS NDVI
     ↓
Vegetation Analysis
     ↓
Dryness Analysis
(Precipitation + ET)
     ↓
Mars-like Index
     ↓
Mars-like Environment Map
```
