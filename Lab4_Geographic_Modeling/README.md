# Lab 4 – Geographic Modeling

## การสร้างแบบจำลองความเสี่ยงน้ำท่วมในจังหวัดสุโขทัย

---

# 1. บทนำ

น้ำท่วมเป็นหนึ่งในภัยพิบัติทางธรรมชาติที่เกิดขึ้นบ่อยในประเทศไทย โดยเฉพาะในพื้นที่ราบลุ่มที่มีลักษณะภูมิประเทศเอื้อต่อการสะสมของน้ำ การวิเคราะห์พื้นที่เสี่ยงน้ำท่วมจึงมีความสำคัญต่อการวางแผนการใช้ที่ดิน การบริหารจัดการภัยพิบัติ และการลดผลกระทบที่อาจเกิดขึ้นกับชุมชนและโครงสร้างพื้นฐาน

งานศึกษานี้มีวัตถุประสงค์เพื่อสร้าง **แบบจำลองเชิงพื้นที่ (Spatial Model)** สำหรับประเมินความเสี่ยงน้ำท่วมใน **จังหวัดสุโขทัย** โดยใช้วิธี **Multi-Criteria Spatial Modeling** ซึ่งเป็นการรวมปัจจัยด้านสิ่งแวดล้อมหลายตัวที่มีผลต่อการเกิดน้ำท่วมเข้าด้วยกัน

แบบจำลองถูกพัฒนาด้วย **Google Earth Engine (GEE)** และใช้วิธี **Weighted Linear Combination (WLC)** เพื่อรวมค่าปัจจัยต่าง ๆ ให้ได้แผนที่ดัชนีความเสี่ยงน้ำท่วม (Flood Risk Index)

---

# 2. พื้นที่ศึกษา

พื้นที่ศึกษาคือ **จังหวัดสุโขทัย ประเทศไทย** ซึ่งตั้งอยู่ในภาคเหนือของประเทศ พื้นที่ส่วนใหญ่เป็นที่ราบลุ่มตามลุ่มน้ำยม จึงมีความเสี่ยงต่อการเกิดน้ำท่วมในช่วงฤดูฝน โดยเฉพาะในพื้นที่ที่มีระดับความสูงต่ำและอยู่ใกล้แม่น้ำ

ขอบเขตพื้นที่ศึกษาได้มาจากฐานข้อมูลขอบเขตการปกครอง **FAO GAUL dataset**

# 3. ข้อมูลที่ใช้ในการศึกษา

ข้อมูลที่นำมาใช้ในการสร้างแบบจำลองมีดังนี้

# 4. วิธีการศึกษา

## 4.1 การสร้างแบบจำลองเชิงพื้นที่

การศึกษานี้ใช้วิธี **Multi-Criteria Spatial Modeling** ซึ่งเป็นการรวมปัจจัยหลายตัวที่มีผลต่อความเสี่ยงน้ำท่วม

ปัจจัยที่ใช้ในแบบจำลอง ได้แก่

* ปริมาณฝน (Rainfall)
* ระดับความสูง (Elevation)
* ความลาดชัน (Slope)
* ระยะห่างจากแม่น้ำ (Distance to river)

ก่อนนำมารวมกัน ปัจจัยทั้งหมดถูก **Normalize** ให้อยู่ในช่วงค่า **0 ถึง 1**

---

## 4.2 ผลกระทบของปัจจัยต่อความเสี่ยงน้ำท่วม

ปัจจัยแต่ละตัวมีบทบาทต่อการเกิดน้ำท่วมแตกต่างกัน ตารางต่อไปนี้แสดงความสัมพันธ์ระหว่างปัจจัยที่ใช้ในแบบจำลองกับระดับความเสี่ยงน้ำท่วม

| ปัจจัย                                | แนวโน้มต่อความเสี่ยงน้ำท่วม   | เหตุผลทางภูมิศาสตร์                                                                           |
| ------------------------------------- | ----------------------------- | --------------------------------------------------------------------------------------------- |
| Rainfall                 | ปริมาณฝนสูงส่งผลต่อความเสี่ยงสูง   | ฝนที่ตกหนักทำให้ปริมาณน้ำสะสมในพื้นที่เพิ่มขึ้น และอาจเกินความสามารถในการระบายน้ำของพื้นที่   |
| Elevation            | พื้นที่ต่ำส่งผลต่อความเสี่ยงสูง    | พื้นที่ที่มีระดับความสูงต่ำมักเป็นพื้นที่รับน้ำหรือพื้นที่สะสมของน้ำ ทำให้เกิดน้ำท่วมได้ง่าย  |
| Slope       | ความลาดชันต่ำส่งผลต่อความเสี่ยงสูง | พื้นที่ที่มีความลาดชันต่ำทำให้น้ำไหลออกจากพื้นที่ช้าลง ส่งผลให้เกิดการสะสมของน้ำ              |
| Distance to River | ใกล้แม่น้ำส่งผลต่อความเสี่ยงสูง    | พื้นที่ที่อยู่ใกล้แม่น้ำมีโอกาสได้รับผลกระทบจากน้ำล้นตลิ่งหรือการเพิ่มขึ้นของระดับน้ำในแม่น้ำ |


---

## 4.3 การกำหนดน้ำหนักของปัจจัย

แบบจำลองใช้วิธี **Weighted Linear Combination (WLC)** โดยกำหนดน้ำหนักของปัจจัยดังนี้

| ปัจจัย            | น้ำหนัก |
| ----------------- | ------- |
| Rainfall          | 0.30    |
| Elevation         | 0.25    |
| Slope             | 0.20    |
| Distance to river | 0.25    |

สมการของแบบจำลองคือ

Flood Risk =
0.30 × Rainfall +
0.25 × Elevation +
0.20 × Slope +
0.25 × Distance to river

จากตารางจะเห็นได้ว่าปัจจัยทั้งหมดมีความสัมพันธ์โดยตรงกับโอกาสเกิดน้ำท่วม 
โดยเฉพาะปริมาณฝนและระยะห่างจากแม่น้ำซึ่งเป็นปัจจัยสำคัญที่กำหนดความเสี่ยงในพื้นที่ลุ่มน้ำ

# 5. ผลลัพธ์ของแบบจำลอง

ผลลัพธ์ของแบบจำลองคือ **Flood Risk Index** ซึ่งมีค่าระหว่าง **0 ถึง 1**

ค่าที่สูงแสดงถึงพื้นที่ที่มีความเสี่ยงน้ำท่วมมากกว่า

เพื่อให้ตีความผลลัพธ์ได้ง่าย ค่า Flood Risk Index ถูกแบ่งออกเป็น **4 ระดับความเสี่ยง**

| ระดับ | ความเสี่ยง        |
| ----- | ----------------- |
| 1     | ความเสี่ยงต่ำ     |
| 2     | ความเสี่ยงปานกลาง |
| 3     | ความเสี่ยงสูง     |
| 4     | ความเสี่ยงสูงมาก  |

# 6. การวิเคราะห์ความไวของแบบจำลอง (Sensitivity Analysis)

เพื่อประเมินความไวของแบบจำลอง ได้ทำการปรับน้ำหนักของปัจจัย **Rainfall** เพิ่มขึ้นและลดลง **±20%**

จากนั้นนำมาคำนวณแบบจำลองใหม่เพื่อดูว่าผลลัพธ์มีการเปลี่ยนแปลงมากน้อยเพียงใด

ผลการทดลองพบว่า

* พื้นที่เสี่ยงหลักยังคงปรากฏในบริเวณเดิม
* การเปลี่ยนแปลงของน้ำหนักปัจจัยทำให้ค่าความเสี่ยงเปลี่ยนเล็กน้อยในบางพื้นที่

แสดงให้เห็นว่าแบบจำลองมี **ความเสถียร (Robustness)** ในระดับหนึ่ง

---

# 7. การตรวจสอบความถูกต้องของแบบจำลอง (Validation)

เนื่องจากไม่มีข้อมูลน้ำท่วมจริงในช่วงเวลาเดียวกัน จึงใช้ **JRC Global Surface Water dataset** เป็นตัวแทนสำหรับการตรวจสอบแบบจำลอง

พื้นที่ที่มีความเสี่ยงสูงในแบบจำลองควรปรากฏใกล้กับพื้นที่ที่มีการเกิดน้ำหรือแม่น้ำถาวร

ผลการตรวจสอบเบื้องต้นพบว่า พื้นที่เสี่ยงสูงหลายพื้นที่อยู่ใกล้กับลำน้ำหลักของจังหวัดสุโขทัย ซึ่งสอดคล้องกับหลักการทางภูมิศาสตร์

---

# 8. ข้อจำกัดของแบบจำลอง

แม้ว่าแบบจำลองนี้สามารถระบุพื้นที่เสี่ยงได้ในระดับหนึ่ง แต่ยังมีข้อจำกัดบางประการ เช่น

* ไม่ได้รวมปัจจัยด้านการใช้ที่ดิน
* ไม่มีข้อมูลน้ำท่วมจริงสำหรับการตรวจสอบความถูกต้อง
* ความละเอียดของข้อมูลฝนต่ำกว่าปัจจัยอื่น

ในอนาคตควรเพิ่มข้อมูลเพิ่มเติม เช่น

* Land use
* Soil type
* River discharge

เพื่อปรับปรุงความแม่นยำของแบบจำลอง

# 9. การตอบคำถามท้ายแลป

## 1. หากมีข้อมูล Ground Truth จริง ความแม่นยำของโมเดลน่าจะอยู่ในระดับใด

คาดว่าความแม่นยำของโมเดลจะอยู่ในระดับ **ปานกลางถึงค่อนข้างสูง** เนื่องจากปัจจัยหลักที่มีผลต่อการเกิดน้ำท่วมได้ถูกนำมาพิจารณา อย่างไรก็ตาม การขาดข้อมูลด้านการใช้ที่ดินและระบบระบายน้ำอาจทำให้ความแม่นยำลดลงในบางพื้นที่

---

## 2. ปัจจัยใดที่ไม่ได้ถูกนำมาใช้แต่มีความสำคัญ

ปัจจัยที่สำคัญแต่ไม่ได้รวมในโมเดล ได้แก่

* การใช้ที่ดิน (Land use)
* ชนิดของดิน (Soil type)
* ความสามารถในการระบายน้ำ
* โครงสร้างพื้นฐานด้านการระบายน้ำ

ปัจจัยเหล่านี้ไม่ได้รวมไว้เนื่องจากข้อจำกัดของข้อมูลในช่วงเวลาการศึกษา

---

## 3. โมเดลนี้เหมาะสมกับการใช้ในระดับพื้นที่ใด

โมเดลนี้เหมาะสมสำหรับการใช้ในระดับ **จังหวัดหรืออำเภอ** เนื่องจากใช้ข้อมูลภูมิประเทศและข้อมูลฝนในระดับภูมิภาค หากใช้ในระดับตำบลหรือชุมชนอาจต้องใช้ข้อมูลที่ละเอียดมากขึ้น

---

## 4. หากต้องอัปเดตโมเดลทุกปี ขั้นตอนใดต้องทำซ้ำ

ขั้นตอนที่ต้องทำซ้ำ ได้แก่

* การอัปเดตข้อมูลฝน
* การคำนวณ Flood Risk Index ใหม่
* การตรวจสอบความไวของแบบจำลอง

ส่วนขั้นตอนที่ไม่ต้องเปลี่ยน ได้แก่

* โครงสร้างของแบบจำลอง
* สมการการรวมปัจจัย
