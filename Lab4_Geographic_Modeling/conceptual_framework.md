# Conceptual Framework

## กรอบแนวคิดของแบบจำลองความเสี่ยงน้ำท่วมในจังหวัดสุโขทัย

---

# 1. คำถามวิจัย (Research Question)

พื้นที่ใดในจังหวัดสุโขทัยมีความเสี่ยงต่อการเกิดน้ำท่วมสูง โดยพิจารณาจากปัจจัยด้านสิ่งแวดล้อม ได้แก่ ปริมาณฝน ระดับความสูงของพื้นที่ ความลาดชัน และระยะห่างจากแม่น้ำ

คำถามนี้สามารถตอบได้โดยการสร้าง **แบบจำลองเชิงพื้นที่ (Spatial Model)** เพื่อแสดงการกระจายตัวของความเสี่ยงน้ำท่วมในรูปแบบของแผนที่

---

# 2. แนวคิดของแบบจำลอง

การเกิดน้ำท่วมไม่ได้เกิดจากปัจจัยเพียงตัวเดียว แต่เกิดจากการรวมกันของหลายปัจจัยทางกายภาพ เช่น

* ปริมาณฝน
* ลักษณะภูมิประเทศ
* ความลาดชัน
* ความใกล้ชิดกับแม่น้ำ

ดังนั้น การศึกษานี้จึงใช้วิธี **Multi-Criteria Spatial Modeling** ซึ่งเป็นการรวมปัจจัยหลายตัวเข้าด้วยกันเพื่อประเมินระดับความเสี่ยง

แบบจำลองใช้วิธี **Weighted Linear Combination (WLC)** ในการรวมค่าของแต่ละปัจจัย

---

# 3. ปัจจัยที่ใช้ในแบบจำลอง

| ปัจจัย            | ความสัมพันธ์กับน้ำท่วม        | เหตุผล                                                          |
| ----------------- | ----------------------------- | --------------------------------------------------------------- |
| Rainfall          | ฝนมาก → ความเสี่ยงสูง         | ปริมาณฝนที่สูงทำให้เกิดการสะสมของน้ำและเพิ่มโอกาสการเกิดน้ำท่วม |
| Elevation         | พื้นที่ต่ำ → ความเสี่ยงสูง    | พื้นที่ที่มีระดับความสูงต่ำมักเป็นพื้นที่รับน้ำ                 |
| Slope             | ความลาดชันต่ำ → ความเสี่ยงสูง | น้ำไหลออกจากพื้นที่ช้า                                          |
| Distance to River | ใกล้แม่น้ำ → ความเสี่ยงสูง    | มีโอกาสเกิดน้ำล้นตลิ่ง                                          |

---

# 4. การกำหนดน้ำหนักของปัจจัย

น้ำหนักของแต่ละปัจจัยถูกกำหนดตามความสำคัญทางภูมิศาสตร์และความสัมพันธ์กับการเกิดน้ำท่วม

| ปัจจัย            | น้ำหนัก |
| ----------------- | ------- |
| Rainfall          | 0.30    |
| Elevation         | 0.25    |
| Slope             | 0.20    |
| Distance to River | 0.25    |

น้ำหนักทั้งหมดรวมกันเท่ากับ 1 เพื่อให้สามารถนำไปใช้ในสมการแบบ Weighted Linear Combination

---

# 5. สมการของแบบจำลอง

Flood Risk Index คำนวณจากสมการ

Flood Risk =
0.30 × Rainfall

* 0.25 × Elevation
* 0.20 × Slope
* 0.25 × Distance to River

ค่าที่ได้จะอยู่ในช่วง **0 – 1** ซึ่งแสดงระดับความเสี่ยงของพื้นที่

---

# 6. การ Normalize ข้อมูล

เนื่องจากข้อมูลแต่ละประเภทมีหน่วยและช่วงค่าที่แตกต่างกัน จึงต้องทำ **Normalization** ก่อนนำมารวมในแบบจำลอง

วิธีที่ใช้คือ **Min-Max Normalization**

Normalized Value = (X − Min) / (Max − Min)

ทำให้ค่าของทุกปัจจัยอยู่ในช่วง **0 ถึง 1**

---

# 7. การตรวจสอบแบบจำลอง (Validation)

การตรวจสอบแบบจำลองทำโดยการเปรียบเทียบพื้นที่ที่แบบจำลองระบุว่าเป็นพื้นที่เสี่ยงสูงกับข้อมูล **JRC Global Surface Water**

หากแบบจำลองมีความสมเหตุสมผล พื้นที่เสี่ยงสูงควรปรากฏใกล้กับพื้นที่ที่มีแม่น้ำหรือแหล่งน้ำถาวร

---

# 8. การวิเคราะห์ความไวของแบบจำลอง (Sensitivity Analysis)

เพื่อประเมินความไวของแบบจำลอง ได้ทำการปรับน้ำหนักของปัจจัย **Rainfall** เพิ่มขึ้นและลดลง **±20%**

จากนั้นนำมาคำนวณแบบจำลองใหม่และเปรียบเทียบผลลัพธ์กับโมเดลเดิม

การวิเคราะห์นี้ช่วยให้เข้าใจว่าแบบจำลองมีความไวต่อการเปลี่ยนแปลงของปัจจัยมากเพียงใด

---
